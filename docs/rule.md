# Bluff Auction — ゲームルール

## 概要

- 4人固定のカードオークション
- 勝利条件: **4種類すべてのアイテムを所有**した最初のプレイヤーが即勝利

## 構成

- カード: 4ブランド(絵画/彫刻/陶器/宝飾)× 各4枚 = 16枚
- 通貨: $(ドル)
- 手札: 所持カード全て(初期配布 + 落札で入手したもの、いずれも出品対象)

## 準備

- 4ブランドを4人に1:1でランダム割当、各自に秘匿
- 自ブランドのカード4枚を裏向きで配布
- 初期所持金 **$100**
- 親を時計回りで決定

## ターン進行

- 1ターン = 1人が出品、時計回りで交代
- 誰かが4種類揃えた時点で即終了

## 出品

- 手札から1枚を裏向きで提示、種別を宣言
  - 実種別と一致・不一致は自由
- **フェイク販売**(自ブランド以外を宣言)は1人ゲーム中**最大2回**
  - 残回数は全員に公開
  - 流札時は消費なし、落札成立で確定
- **初期落札額**を設定(出品者の所持金以下)

## 入札(公開競り上げ)

- 出品者以外が参加、任意額を宣言して競り上げ
- 入札額は全員に公開
- パスで離脱、1人残った時点で落札確定
- **流札時**(全員パス)
  - カードは出品者の手札に戻る
  - 出品者は初期落札額を他プレイヤー全員に均等分配
  - 端数は出品者吸収

## 取引成立

- 落札者 → 出品者へ落札額を支払い
- カードは落札者の手札へ裏向き保管(再出品可)
  - 落札者のみ真の種別を確認可能
- 落札者の開示・秘匿・虚偽発言は自由

## 勝敗判定

- **即時勝利**: 4種揃えた最初のプレイヤー
